# Program Faktorial Bilangan (Java)

# Deskripsi
Program ini digunakan untuk menghitung **faktorial dari bilangan bulat positif** yang dimasukkan oleh pengguna.  
Program menggunakan **BigInteger** agar dapat menghitung faktorial dari bilangan yang sangat besar.

# Cara Kerja Program
1. Pengguna memasukkan bilangan bulat positif.  
2. Program memeriksa apakah input valid (bukan huruf atau bilangan negatif).  
3. Jika valid, program menghitung faktorial menggunakan perulangan.  
4. Hasil faktorial ditampilkan di layar.
